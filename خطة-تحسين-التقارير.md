# خطة تحسين نظام التقارير — HBOperations

## الوضع الحالي

| العنصر | الحالة |
|--------|--------|
| صفحة التقارير (`Reports.razor`) | ❌ محذوفة — الرابط `/reports` يُعطي 404 |
| API تصدير Excel (`/api/reports/branches/export`) | ✅ موجود — تقرير فروع فقط |
| API تصدير PDF (`/api/reports/branches/export-pdf`) | ✅ موجود — QuestPDF بتصميم بسيط (جدول فقط) |
| الإحصائيات (Dashboard) | ✅ موجودة في `Home.razor` — عامة وغير مخصصة بشكل كافٍ |
| تخصيص حسب الدور | ⚠️ جزئي — فلترة البيانات موجودة لكن الواجهة واحدة للجميع |

---

## الرؤية المستهدفة

> **صفحة تقارير احترافية** بإحصائيات بصرية مخصصة لكل مستخدم حسب دوره، مع إمكانية تصدير تقارير PDF بتصميم يطابق هوية البنك (HTML+CSS → PDF)، وتصدير Excel للبيانات الخام.

## التقارير حسب الدور

| الدور | المحتوى المعروض |
|-------|-----------------|
| **SuperAdmin / CEO / ITAdmin** | إحصائيات كل الفروع + كل الإدارات + أداء عام + مقارنة فروع |
| **AssistantCEO / Auditor / Compliance** | نفس الأعلى (قراءة فقط) |
| **DepartmentManager** | إحصائيات إدارته + أداء موظفيه + معاملات الإدارة |
| **BranchManager / OfficeManager** | إحصائيات فرعه + أداء موظفيه + معاملات الفرع |
| **DepartmentStaff** | ملخص معاملاته الشخصية (مُرسَلة/مستلَمة/معلّقة) |
| **BranchStaff** | ملخص معاملاته الشخصية |
| **الشؤون الإدارية** | إحصائيات التوصيل + متوسط وقت التسليم + المعاملات المعلقة |

## تصدير PDF — التصميم

### المكتبة المُختارة: **QuestPDF** (موجودة بالفعل)

> QuestPDF تدعم تصميم مخصص بالكامل (ألوان، جداول، headers/footers، شعار) — لا نحتاج HTML→PDF لأن QuestPDF أقوى وأسرع وأجمل.

### هيكل ملف PDF المُصدَّر

```
│  ║  ─────────────────────────────────────── ║   │
│  ║  التاريخ: YYYY/MM/DD  │  أُعد بواسطة: X  ║   │

│                                                  │
│  ┌─────────────────────────────────────────┐    │
```

## أنواع التقارير المتاحة

### 1. تقرير الفروع (SuperAdmin/CEO)
- **المحتوى**: أداء كل فرع (صادرة/واردة/مستلمة/معلقة/مرفوضة)
- **PDF**: جدول + ملخص بصري
- **Excel**: بيانات كاملة قابلة للتحليل

### 2. تقرير الإدارات (SuperAdmin/CEO/DeptManager)
- **المحتوى**: أداء كل إدارة + عدد المعاملات
- **PDF**: جدول + مقارنة
- **Excel**: تفاصيل كل إدارة

### 3. تقرير الأداء (الكل — مخصص حسب الدور)
- **المحتوى**: متوسط وقت المعالجة، نسبة الإنجاز، أسرع/أبطأ
- **PDF**: مؤشرات KPI + جدول
- **Excel**: بيانات زمنية

### 4. تقرير المعاملات التفصيلي (Staff+)
- **المحتوى**: قائمة المعاملات مع الفلاتر المطبقة
- **PDF**: جدول مفصل (رقم مرجعي، نوع، مرسل، مستلم، حالة، تاريخ)
- **Excel**: بيانات كاملة

### 5. تقرير الشؤون الإدارية (إدارة الشؤون)
- **المحتوى**: معاملات تم استلامها/تسليمها، متوسط زمن التوصيل
- **PDF**: ملخص + KPI
- **Excel**: تفاصيل


### المكتبات المستخدمة

| المكتبة | الاستخدام | الحالة |
|---------|-----------|--------|
| **QuestPDF** 2025.1.2 | توليد PDF احترافي | ✅ موجودة |
| **ClosedXML** 0.105 | تصدير Excel | ✅ موجودة |
| **CSS Charts** (Pure CSS) | رسوم بيانية في الصفحة | ✅ موجود (نمط Home.razor) |
